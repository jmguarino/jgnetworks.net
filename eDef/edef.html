EarthDefender

This is a small space shooter I'm working on. It's written in libGDX whjch is a Java game engine library that can target Android, PC, iOS, and HTML5. I made all of the assets so please excuse the programmer art.

Note: The game is not designed to work on small format screens so it likely will not load if you are on a mobile device, otherwise it should load in a few seconds.

↻

Mouse Controls

  • Drag left/right to move ship.
  • Left click to shoot.
  • Click speaker to toggle sound.

Keyboard Controls

  • Left/Right Arrows for movement.
  • Space to shoot.
  • M to toggle sound.
